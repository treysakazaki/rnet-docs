rnet.VertexData#

class rnet.VertexData(df, layer=None)#

Class for representing vertex data.

Vertices are the points used to define the geometry of each road.

Parameters

  • df (pandas.DataFrame) – Frame summarizing vertex dataset. with index ‘id’ and columns [‘x’, ‘y’] for two-dimensional vertices, or [‘x’, ‘y’, ‘z’] for three-dimensional coordinates.

  • layer (VertexLayer, optional) – Layer for rendering vertex features.
