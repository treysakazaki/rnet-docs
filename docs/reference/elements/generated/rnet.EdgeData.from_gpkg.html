rnet.EdgeData.from_gpkg#

classmethod EdgeData.from_gpkg(gpkg, layername='edges')#

Instantiates EdgeData from a GeoPackage layer.

Parameters

  • gpkg (str or GpkgData) – Path or GpkgData instance specifying the GeoPackage to read.

  • layername (str, optional) – Name of the layer to read. The default is ‘edges’.

Return type

EdgeData

Raises

  • TypeError – If gpkg is unexpected type.

  • LayerNotFoundError – If layer with the specified name does not exist.
